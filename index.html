HTML AND CSS

The Basics

What is HTML and CSS?

  • HTML and CSS are languages for web development
  • HTML stands for Hyper Text Markup Language, while CSS stands for Cascading Style Sheets
  • HTML and CSS are very easy to learn, while providing a whole new set of things to do
  • Both languages are used together to create a website

HTML

A picture of coffee. The image was added using HTML and it was given an oval border using CSS.

HTML is the skeleton of a website. It tells your browser what to display.

You must learn HTML to create a website.

With HTML, you can display text, fonts, images, hyperlinks, and more.

CSS

CSS is used to add decoration and design to websites.

You don't need to learn CSS to create a website, but without CSS, it would look bland.

Use CSS to change fonts, add colour, and add flare to images.

Terminology

Hyperlinks

Hyperlinks are an important element in web development. They are clickable elements that open a URL to another website.

URL

URLs, also known as links, stand for Uniform Resource Locator. They are used as an address for websites, much like how your home has an address.

Extra Info

JavaScript

JavaScript is another coding language mostly used in web development. It adds functionality to websites, such as creating clickable buttons, doing calculations, and more.
